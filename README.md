# Privacy Policy - WirelessUSB Transfer

Last updated: March 16, 2026

WirelessUSB Transfer app does not collect, store, or share 
any personal data. The app only accesses your local WiFi 
network to transfer files between your Android device and PC.

## Data Collection
This app does NOT collect any personal information.

## Network Access
App uses WiFi only for local file transfer. No internet 
connection is required or used.
